Methodology & caveats
  • Cohort: 425 MBG-eligible cspids. Banner clicks = banner_opened events in CLEVERTAP_CSP_API.EVENTS_DATA, mapped CLEVERTAP_ID → cspid via PROFILE_DATA (each CT id → exactly one cspid, no fan-out).
  • Install funnel: DBT_CSP.TAS_INSTALL_EXECUTION_CANDIDATES (ETL_CURRENT=TRUE). Slot = CONFIRMED_SLOT_AT, Technician = EXECUTOR_ID, Installed = OTP_VERIFIED / COMPLETED_STEP≥7, joined on CSP_ID.
  • Non-MBG banner base is small because the banner is live almost exclusively to the 425 — the per-connection output funnel is the fair network benchmark.
  • Immature window: most connections are still in flight, so absolute install counts keep rising; treat rates as directional. Data auto-refreshes every hour (Metabase → Snowflake).
